# 文本格式

Invero 在容器 GUI 属性和相关物品的显示属性，以及脚本中的语言输出中  
都支持一定的文本格式化处理方式

按照处理顺序先后，依次为

### Kether Inline

- 脚本内联函数
- `{{identifier}}`
-

### Placeholder API

- `%identifier%`

### MiniMessage

- 当且仅当你的 **服务端** 包含新版的 MiniMessage 时此功能才有效 （Paper 1.16.5+）
- 查阅文档 [MiniMessage Format](https://docs.adventure.kyori.net/minimessage/format.html#standard-tags)
